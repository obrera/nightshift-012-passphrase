# nightshift-012-passphrase

A small browser-based passphrase generator built with TypeScript + React + Vite + Tailwind.

## Features

- Generate multi-word memorable passphrases locally
- Configure word count and separator
- Optional capitalization, 2-digit number, and symbol suffix
- Estimated entropy indicator
- One-click clipboard copy

## Stack

- TypeScript
- Bun
- Vite + React
- Tailwind CSS

## Run locally

```bash
bun install
bun dev
```

Build for production:

```bash
bun run build
```
